# Installation

`isodata` supports python 3.7+. Install with pip

```
python -m pip install isodata
```

Upgrade using the following command

```
python -m pip install --upgrade isodata
```

Check current version like this

```{code-cell}
:tags: [my-tag]

import isodata
isodata.__version__
```
